Documentação do Projeto Aromê utilizando TailWind

  1. Criei a pasta 'Arome', onde vai ficar todo o projeto
  2. Decide utilizar o boilerplate do 'Vite' para criação desse projeto, então entrei no site do Vite e copiei o seguinte comando 'npm create vite@latest'
  3. Utilizando o terminal do VSCode entrei na pasta 'Arome' e rodei o comando 'npm create vite@latest'
  4. Coloquei 'arome-ds' como o nome do projeto
    1. Selecionei o framework 'React'
    2. Selecionei a variante 'Typescript'
  5. Entrei na pasta 'arome-ds' e rodei 'npm install' para instalar as dependencias
  6. Apaguei todos os arquivos CSS e a pasta 'assets'
  7. Deixei somente a carcaça do arquivo 'App.tsx'
  8. Em 'app.tsx' troquei 'export defaul App' por 'export function App()'
  9. Dividi o terminal do VSCODE em dois e digitei no segundo terminal: npm install -D tailwindcss postcss autoprefixer
    • OBS: Instalei como dependencia de desenvolvimento

    • tailwindcss: É o próprio Tail Wind
    • postcss: É um bundler para CSS, ele automatiza tarefas dentro do CSS. Basicamente ele fica lendo os arquivos e criando o CSS necessário para o TailWind funcionar na nossa página
    • autoprefixer: É uma biblioca que adiciona prefixers para algumas funcionalidades do CSS que só funcionam em alguns brownser específicos
  10. Digitei no segundo terminal: npx tailwindcss init -p
    • O '-p' é para ele também criar o arquivo de configuração do postcss
    • O Vite já tem integração automática com o 'postcss', então quando ele detecta que tem um arquivo 'postcss' na raíz do projeto ele automaticamente já começa a entender
  11. No arquivo 'tailwind.config.js' passei para o Tailwind onde estão os arquivos de conteúdo, ou seja, onde vão estar os arquivos que vão ter classes do tailwind
    • Passei: './src/**/*.tsx',
      1. ** : Quer dizer 'qualquer pasta'
      2. *.tsx : Quer dizer 'qualquer arquivo terminado em .tsx'
  12. Criei uma pasta 'styles' dentro da 'src'
  13. Dentro da pasta 'styles' criei um arquivo 'global.css'
  14. Instalei as seguintes extensões no VSCode:
    • Tailwind CSS IntelliSense
    • PostCSS Language Support
  15. No arquivo 'global.css' importei 3 conjuntos de estilizações que o TailWind traz
    • @tailwind base;
    • @tailwind utilities;
    • @tailwind components;
  16. Importei o arquivo global.css no arquivo App.tsx
    • A partir de agora é possível ver que o 'Hello World' está sem estilização
