Documentação - Projeto AROMÊ

Utilizando Vite, React e TypeScript

  1. Entrei em https://git-scm.com/download/win, baixei o Git e logo em seguida o instalei no Windows
  2. Abri o terminal CMD do Windows e digitei git para terminar a instalação do Git
  3. Ainda no terminal configurei o git com o meu nome e e-mail através de git config --global user.name "Tierres Griep" e git config --global user.email "tierresgriep@gmail.com"
  4. No site https://nodejs.org/en/download/current baixei o NodeJS e logo após o instalei
  5. Abri o CMD e instalei o Vite globalmente com: npm install -g create-vite
    • O NPM vem junto ao instalar o NodeJS
  6. Dentro da pasta Arome rodei npm create vite@latest e configurei o projeto da seguinte maneira:
    • Project name: arome-ds
    • Select a framework: React
    • Select a variant: TypeScript
  7. Entrei na pasta arome-ds e rodei npm install para terminar de criar o esqueleto do projeto(assim como manda o Vite)
  8. Rodei npm run dev para abrir o servidor do projeto na minha máquina
  9. Apaguei todos os arquivos CSS e a pasta 'assets'
  10. Deixei somente a carcaça do arquivo 'App.tsx'
  11. Em 'App.tsx' troquei 'export defaul App' por 'export const App = () => {}'
  12. Criei as pastas components e screens
  13. Dentro de components criei uma pasta para cada componente que formam as páginas e criei uma pasta 'shared' onde ficam os componentes que são compartilhados entre as páginas
  14. Dentro de screens criei uma pasta para cada página principal
    • Essas páginas utilizarão os componentes para formar a página em si
  15. Instalei a biblioteca React-Router-Dom-V6 com o seguinte comando: npm install react-router-dom --save
  16. Todos os componentes são do tipo function
  17. Eu utilizei o seguinte padrão para todos os componentes:
    • export const Header = () => {}
  18. Git e GitHub
    1. Iniciei o Git no projeto digitando "git init" na pasta "PROJETOS/Arome"
    2. Adicionei todos os arquivos do projeto para a área de staging com git add .
    3. Comitei todo o projeto com git commit -m 'My first commit'
    4. Linkei o meu projeto do PC com um repositório do Github com o comando git remote add origin https://github.com/tierres/Arome.git
    5. Subi de fato o meu projeto pela primeira vez para o repositório no Github com git push origin master
    6. No meu notebook utilizei git remote add origin https://github.com/tierres/Arome.git para conectar o diretório do notebook com o diretório da nuvem
    7. A partir de agora sempre que eu quiser baixar no PC ou no notebook o projeto atualizado do Github, vou digitar: git pull origin master
  19. Comecei a construção do site pela screen "Home"
    1. No arquivo Home.tsx coloquei os componentes: Header, SintaOPrazerDoAroma em ordem de exibição
    2. No componente Header criei uma pasta Vetores
      • Nessa pasta adicionei todos os vetores da logo do site que baixei do projeto Figma
    3. No componente Header importei todos os vetores e adicionei eles na renderização
    4. Criei um 'Home.css' para fazer a estilização da página Home e importei no arquivo Home.tsx
    5. Em Home.css criei um estilo para o container do componente Home e importei esse estilo na div que envolve os componentes(em Home.tsx)
